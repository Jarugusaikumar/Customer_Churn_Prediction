# Power BI Dashboard Assets

This folder contains data files and SQL scripts to support a Power BI dashboard for customer churn insights.

## Files
- customer_churn_dashboard_data.csv: sample churn dataset for dashboard visuals
- powerbi_contract_summary.csv: contract-level summary for KPI cards and bar charts
- powerbi_setup.sql: SQL scaffold to create a simple dashboard table

## Suggested visuals
- KPI cards: total customers, churn rate, average monthly charges
- Bar chart: churn rate by contract
- Column chart: average tenure by contract
- Scatter plot: monthly charges vs total charges
